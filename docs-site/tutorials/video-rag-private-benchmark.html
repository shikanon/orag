Legal boundary

No benchmark mirror.

No video, subtitle, annotation, question or answer data is fetched, cached, proxied or published by ORAG.

Private validation

Trust before use.

A private source must have a verified digest, bounded metadata, accepted media type and ordered timed subtitles.

Deterministic

Same source, same proof.

Fixed protocol cadence produces stable segment and evidence identifiers without browser-controlled timestamps.

No fallback

Temporal means temporal.
