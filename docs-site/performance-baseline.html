What is measured

Six honest dimensions.

Ingestion throughput, query p50/p95, cache-hit rate, evaluation duration, model calls and cost all travel with the same provenance record.

What is rejected

No apples-to-oranges.

Unknown fields, real-provider ambiguity, weak sample counts, invalid percentiles and non-recomputable throughput fail verification.
